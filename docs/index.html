Some Facts

  • Interestingly, its name is inspired by the World Wide Web.
  • What's really cool is that it was designed to avoid using keywords from any specific human language.
  • This makes programming potentially accessible to everyone, regardless of their native tongue!
  • It's declarative, meaning you tell it what you want, rather than how to do it.
  • This approach helps bridge the gap between servers and clients in web applications and beyond.
  • It's aiming to fundamentally shift how people program, perhaps making it more intuitive.
  • While being human-centered in its design, it's also deeply focused on the world of Artificial Intelligence.
  • The goal is to be simple, easy to pick up, yet complete and incredibly capable.

Language Type System

Wide itself has no Keywords like most programming languages do.

It's basic type system looks like this:

'' Char (single letter or digit)
"" String (words or text)

Default Values

When the type is passed followed by no assigned value a default value will be automatically assigned.

It's mostly a zero, except for string and object types.

These values will be used until you manually assign another.

` default value is 0 (false)
isRaining:?

` default value is 0 (vazio)
windDirectionInitial:''

` default value is vazio
stationName:""

` default value is 0
stationID:0

` default value is 0.0
currentTemperatureCelsius:0.0

` default value is 0.00
atmosphericCO2ppm:0.00

` default value is <>, also called a 'Fragment'.
station:<>

This is also true for the objects' properties:

station:= <
    isRaining:?
    winDirectionInitial:''
    stationName:""
    stationID:0
    currentTemperatureCelsius:0.0
    atmosphericCO2ppm:0.00
/>

Printing on Screen

That seems nice, but how do you print on screen?

To print on screen you just enclose what you want to print inside double quotes ""

"Hello, Wide!"

Interpolation

You can use curly braces to dinamically output values:

name:= "Wide"
"Hello, {name}!" `outputs: "Hello, Wide!"

price:= 9.99
"It's ${9.99}." `outputs: "It's, $9.99."
